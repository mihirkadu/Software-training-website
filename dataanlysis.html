Foundational topics

  • Introduction to Data Analytics: An overview of the field, the data analysis process (like the OSEMN framework), and the role of a data analyst.

  • Statistics: Descriptive and inferential statistics, probability, sampling, and hypothesis testing.

  • Data Cleaning and Preprocessing: Identifying and handling missing data, data transformation, and data quality checks.

Tools and programming

  • Microsoft Excel: Data formatting, essential formulas, PivotTables, and creating charts.

  • SQL: Database fundamentals, data manipulation (INSERT, SELECT), and querying data from tables.

  • Python: Using libraries like NumPy for numerical operations and Pandas for data handling and manipulation.

  • R Programming: Data manipulation with tools like dplyr, statistical analysis, and visualization with ggplot2.

Visualization and business intelligence

  • Tableau: Creating interactive charts, reports, and dashboards.

  • Power BI: Building visualizations, reports, and dashboards, and using DAX for calculations.

  • Data Visualization Principles: Fundamentals of effective data storytelling and visual communication.

Advanced topics and projects

  • Machine Learning: Introduction to concepts and algorithms like linear/logistic regression and classification trees.

  • Business Intelligence: Connecting data from different sources, data modeling, and creating dashboards for business insights.

  • Capstone Projects: Applying learned skills to solve real-world case studies or build a comprehensive data analysis project.
